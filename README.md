# Sistema CRUD de Gestión de Experimentos Científicos

## Descripción

Este pequeño proyecto consiste en un sistema CRUD desarrollado en Python que permite gestionar un registro de experimentos científicos.
El programa funciona mediante un menú interactivo en consola que permite al usuario visualizar, crear, consultar, actualizar y eliminar registros almacenados en memoria.

El objetivo del proyecto es aplicar conceptos fundamentales de programación en Python como el uso de funciones, estructuras de datos y control de flujo.

---

## Funcionalidades

El sistema permite realizar las siguientes operaciones:

* Ver todos los registros de experimentos
* Crear un nuevo experimento
* Consultar información de un experimento específico
* Actualizar los datos de un experimento existente
* Eliminar un experimento del registro
* Salir del sistema

---

## Tecnologías utilizadas

Este proyecto fue desarrollado utilizando:

* **Python 3**
* Listas para almacenar los registros
* Diccionarios para representar cada experimento
* Funciones para modularizar el código
* Bucles `while` para el menú interactivo
* Condicionales `if/elif` para el control de opciones

---

## Estructura de los datos

Cada experimento se almacena como un diccionario con la siguiente estructura:

```python
{
    "Nombre": "Penicilina",
    "Investigador": "Alexander Fleming",
    "Area": "Biología",
    "Año": 1928,
    "Resultado": "Descubrimiento del primer antibiótico"
}
```

Todos los experimentos se almacenan dentro de una lista que funciona como base de datos temporal.
